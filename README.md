Yuanfei Huang, Jie Li, Xinbo Gao*, Lihuo He and Wen Lu, "Single Image Super-resolution via Multiple Mixture Prior Models", IEEE Transactions on Image Processing (TIP), vol.27, pp.5904-5917, 2018.
